AI-powered customer support chat widget for your WordPress site.

== Description ==

SupportAI brings AI-first customer support to your WordPress site. Our chat widget allows your visitors to get instant answers powered by AI, with seamless handoff to human agents when needed.

**Features:**

* AI-powered first responder that handles common questions
* Beautiful, customizable chat widget
* Seamless human handoff when needed
* Works with Live Chat, Email, and WhatsApp
* Easy setup - just add your Workspace ID
* Mobile responsive design
* Fully customizable colors and branding

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/supportai/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to SupportAI in the admin menu
4. Enter your Workspace ID and enable the widget
5. That's it! The chat widget will appear on your site

== Frequently Asked Questions ==

= Where do I get a Workspace ID? =

Sign up at [app.supportai.com](https://app.supportai.com) and create a workspace. You'll find your Workspace ID in the Settings section.

= Can I customize the widget colors? =

Yes! You can customize the primary color to match your brand in the plugin settings.

= Is this compatible with my theme? =

The widget is designed to work with any WordPress theme. It floats above your content and doesn't interfere with your design.
